判断当前节点有没有左儿子，
有，判断这个左儿子是不是叶节点
是，累加左儿子值

```
class Solution {
    int res = 0;
    public int sumOfLeftLeaves(TreeNode root) {
        if (root == null) {
            return 0;
        }   
        dfs(root);
        return res;
    }
    public void dfs(TreeNode root) {
        if (root == null) {
            return;
        }
        
        if (root.left != null) {
            // 左子树不是空，证明有左儿子
            if (root.left.left == null && root.left.right == null) {
                // root.left是叶节点
                res += root.left.val;
            }
        }
        dfs(root.left);
        dfs(root.right);
        
    }
}
```
