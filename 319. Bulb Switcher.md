是一个数学问题，

求多少个灯泡是亮的，题意：奇数次按钮，灯泡就是亮的。
灯泡x最后会被按多少次？x的约数个数， 如果个数是奇数，就是亮的。
求所有约数个数是奇数的灯泡的个数

约数个数为奇数的性质：
x被分解为：
x = p1^a1 * p2^a2 * ...
约数个数：
(a1+1)*(a2+1)*...*(ak+1)
个数为奇数，意味着括号里每一个都是奇数：
a1+1是奇数，那么a1就是偶数

x=((p1^(a1/2)) * (p2^(a2/2))... )  ^ 2

就演变成求x的平方根了。




```
public class Solution {
    public int bulbSwitch(int n) {
        return (int)Math.sqrt(n);
    }
}
```
